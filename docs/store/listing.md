## 🇬🇧 English (EN)

**Short description** (132 char limit):
```
Download media from WhatsApp Web chats and groups. Date filter, ZIP mode, unlimited.
```

**Detailed description**:

```
WhatsApp Galeri is a modern Chrome extension that lets you download media files from WhatsApp Web chats and groups.

FEATURES
━━━━━━━━

• All media types: images, videos, audio, documents
• Date range filter: pick media from a specific period
• Single ZIP or individual file downloads
• Deep scan: loads older messages not in cache
• Filename customization: add date, keep original name
• Chat statistics: see how many images, videos, audios, documents exist
• Live progress bar and detailed log
• Dark-theme-native design

HOW TO USE
━━━━━━━━━

1. Open https://web.whatsapp.com
2. Click the extension icon
3. Select a chat → click "Download"
4. Media files are downloaded automatically (or packaged as ZIP)

PRIVACY
━━━━━━

• No data collection
• No server connections
• Everything runs locally in the browser
• No Google OAuth, no paid subscription, no third-party APIs

LICENSE: MIT — open source
SUPPORT: GitHub Issues
```
